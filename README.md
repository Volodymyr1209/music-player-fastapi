# Music Player

Навчальний проєкт на FastAPI, створений в межах практичної роботи №1.

## Використані технології

* Python 3.11
* FastAPI
* uv
* Ruff
* pre-commit
* Git

## Запуск проєкту

Створення віртуального середовища:

uv venv

Активація середовища:

.venv\Scripts\activate

Запуск FastAPI:

python -m uvicorn main:app --reload

## Документація API

http://127.0.0.1:8000/docs
